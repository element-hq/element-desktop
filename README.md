# Purple Element Desktop

Purple Element Desktop is a fork of Element Desktop – Matrix client for desktop platforms with Element Web at its core. The difference from the "classic" Element is that the Purple Element is based on I2PD.

# First Steps

Before you do anything else, fetch the dependencies:

```
yarn install
```
